# pc-cns-resource

这是一个用于归档和发布多个独立 HTML 页面的静态资源仓库。页面内容不针对任何特定域名或网站，仅作为通用 HTML 文件的存储与分发用途。

## 项目简介

本仓库主要存放各类独立制作的 HTML 页面，可用于测试、演示、存档或临时分享。所有页面均为纯前端静态文件，不包含后端逻辑或数据库依赖。

## 目录说明

```
pc-cns-resource/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面引用的公共资源（样式、脚本、图片等）
└── README.md       # 本说明文件
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名通常反映页面用途或内容主题。
- `assets/`：存放页面间共享的静态资源，按需要组织子目录。

## 页面归档说明

- 每个 HTML 页面均为完整自包含文件，或仅依赖 `assets/` 目录下的资源。
- 页面内容不涉及任何商业推广、敏感信息或具体域名指向。
- 归档目的在于提供一份可随时访问的 HTML 副本，便于查阅或引用。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件，或通过静态服务器访问。
3. 也可将 `pages/` 目录部署至任意静态托管服务。
